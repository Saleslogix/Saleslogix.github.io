Saleslogix Mobile Platform

Saleslogix mobile, codenamed "argos" started in June of 2010. The entire suite of "argos" projects are open source and hosted on Github. The platform consists of an SDK (argos-sdk) and a core application (argos-saleslogix). The SDK allows for extensive customizations so developers can extend and override without being forced to create an entirely new application.
